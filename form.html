Serveur local avec Python

Il est possible de créer un serveur web local à partir de Python. Il peut s'avérer intéressant d'implémenter un serveur web dans votre application. Cela permet notamment une communication entre les programmes via un navigateur. On créé d'abord un script serveur.py. 

import http.server
PORT = 8888
server_address = ("", PORT)
server = http.server.HTTPServer
handler = http.server.CGIHTTPRequestHandler
handler.cgi_directories = ["/"]
print("Serveur actif sur le port :", PORT)
httpd = server(server_address, handler)
httpd.serve_forever()
